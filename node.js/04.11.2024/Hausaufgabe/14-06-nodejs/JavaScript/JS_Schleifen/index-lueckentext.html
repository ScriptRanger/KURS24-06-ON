Grundlagen zu Schleifen in JavaScript

        Schleifen sind eine essenzielle Programmiertechnik, mit der wir Code mehrmals
        ausführen können. In JavaScript gibt es mehrere Arten von Schleifen, darunter
        for-Schleifen, while-Schleifen und do...while-Schleifen. Jede hat ihren spezifischen
        Anwendungsfall.

        1. for-Schleife

        Die for-Schleife wird am häufigsten dann verwendet, wenn die Anzahl der
        Wiederholungen im Voraus bekannt ist. Sie besteht aus drei Hauptbestandteilen:
        Initialisierung, Bedingung und das Aktualisieren der Schleifenvariablen.

        

        for (let i = 1; i <= 5; i++) {
            // Code der mehrmals ausgeführt wird
            console.log(i);
        }
        


        Die for-Schleife oben gibt die Zahlen von 1 bis 5 in der Konsole aus.

        2. while-Schleife

        Die while-Schleife wird verwendet, wenn die Anzahl der Wiederholungen nicht im
        Voraus bekannt ist. Die Schleife wird solange ausgeführt, wie die Bedingung true ist.

        

        let i = 1;
        while (i <= 5) {
            console.log(i);
            i++;
        }
        


        Die while-Schleife oben gibt ebenfalls die Zahlen von 1 bis 5 in der Konsole aus.

        3. do...while-Schleife

        Die do...while-Schleife ist ähnlich wie die while-Schleife, jedoch wird der Codeblock
        mindestens einmal ausgeführt, bevor die Bedingung geprüft wird.

        

        let i = 1;
        do {
            console.log(i);
            i++;
        } while (i <= 5);
        


        Die do...while-Schleife oben gibt ebenfalls die Zahlen von 1 bis 5 in der Konsole aus.
